# mp3

mp3 decoder written in pure Rust.

*WORK IN PROGRESS, nothing functional yet*

## Use of unsafe

The entire library uses not a single line of unsafe code.
In fact, lib.rs contains the `#![forbid(unsafe_code)]` directive.
